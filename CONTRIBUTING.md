# Contributing to PEACE

First off, thank you for considering contributing to PEACE! 

## 🔑 Prerequisites for Testing
Because PEACE is entirely serverless and runs locally, **you will need a [Gemini API Key](https://aistudio.google.com/)** to test any changes you make to the UI or routing logic. The app will prompt you for this key when you load `index.html`. 

*Note: Your API key is perfectly safe and never leaves your local machine.*

## How to Contribute
1. **Fork the repository** to your own GitHub account.
2. **Create a branch** for your feature or bug fix: `git checkout -b feature/your-cool-feature`
3. **Commit your changes** with clear, descriptive messages.
4. **Push to your branch**: `git push origin feature/your-cool-feature`
5. **Open a Pull Request (PR)** against the `main` branch of this repository.

## Current Priority: The Great Refactor
The main `index.html` file has become too large to easily maintain. If you are looking to help with the modularization/refactoring process, please check the open **Issues** first. 

If you want to propose a new architecture (like moving it to React or splitting it into Vanilla JS modules), please **open an Issue first** to discuss the approach before you spend hours writing code!

## Code of Conduct
Please be respectful and collaborative. We are building a cool tool together!
